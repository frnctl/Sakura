# Sakura

Official multi link site for Sakura Nagatsuka.

Static HTML/CSS/JS, deployed with GitHub Pages from the repository root.

## Assets

- `assets/karioka.jpg` — "Karioka" single cover (on-page featured card)
- `assets/og.jpg` — 1200×630 social share card (Open Graph / Twitter)
- `assets/hero.jpg` — banner photo (Paris night photoshoot)
- `favicon.svg`, `icon-192.png`, `icon-512.png`, `apple-touch-icon.png` — icons / PWA
